# Youssef Eid Said — Developer Portfolio

A modern, dark-themed freelance portfolio website built with pure HTML5, CSS3, and Vanilla JavaScript.

## 📁 Project Structure

```
portfolio/
├── index.html              # Main HTML file
├── assets/
│   ├── css/
│   │   └── style.css       # All styles (CSS variables, responsive, animations)
│   ├── js/
│   │   └── main.js         # Custom cursor, terminal, scroll effects, form
│   └── images/             # Add your images here
└── README.md
```

## ✨ Features

- Dark developer aesthetic with JetBrains Mono + Syne fonts
- Animated terminal in hero section
- Custom cursor with hover effects
- Intersection Observer scroll animations
- Active nav link highlighting
- Contact form with validation
- Fully responsive (mobile, tablet, desktop)
- Noise texture overlay for depth
- SEO-friendly semantic HTML
- Zero dependencies (no React, Vue, etc.)
- Bootstrap Icons via CDN

## 🚀 Usage

Simply open `index.html` in any modern browser. No build step required.

To deploy: Upload all files to any static hosting (Netlify, GitHub Pages, Vercel, etc.)

## 🎨 Customization

All colors are CSS variables in `style.css`:
- `--accent`: #00ff88 (green)
- `--bg`: #080808 (near black)
- Change fonts in `<head>` Google Fonts link
